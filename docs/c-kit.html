5つの便利機能

🔍

Link Checker

ページ内のリンクエラーを検出

  • 404/500エラーを自動検出
  • リンクを色分けして表示
  • 詳細な統計レポート
🔐

URL Decode/Encode

URLのエンコード・デコード

  • ワンクリックで変換
  • キーボードショートカット対応
  • クリップボードにコピー
✨

Code Formatter

コードを整形・圧縮

  • JSON/HTML/CSS/JSに対応
  • シンタックスハイライト
  • Minify機能搭載
🔄

Domain Switcher

URLペアを簡単に切り替え

  • 開発/本番環境の切り替え
  • URLペアの登録・管理
  • 新しいタブで開くオプション
📊

Embedded Checker

埋め込みコードを検出

  • 埋め込みstyle/scriptタグを検出
  • 外部ファイル化を推奨
  • 詳細レポート表示

インストール方法

  1. GitHubリポジトリからC-kitフォルダをダウンロード
    git clone https://github.com/shimomura2020goout/claude-first-pjts.git
  2. Chromeで chrome://extensions/ を開く
  3. 右上の「デベロッパーモード」をONにする
  4. 「パッケージ化されていない拡張機能を読み込む」をクリック
  5. C-kitフォルダを選択
  6. 拡張機能が読み込まれ、すぐに使用可能になります！
