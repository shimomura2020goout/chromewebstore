インストール方法

🛡️ 安全なインストール

セキュリティ保護のため、Chrome Web Storeからのみインストールしてください。

Chrome Web Store経由のインストールにより、Googleの審査済みバージョンを安全にご利用いただけます。

Chrome Web Storeからインストール

  1. Chrome Web StoreでC-kitを検索
    🔗 Chrome Web Storeで検索 →
  2. 「Chromeに追加」ボタンをクリック
  3. 確認ダイアログで「拡張機能を追加」をクリック
  4. 拡張機能アイコンをクリックしてサイドパネルが開きます！
⚠️ セキュリティに関する重要なお知らせ

GitHubなど他のソースからのインストールは、セキュリティリスクがあるため推奨しません。 必ずChrome Web Store経由でインストールしてください。

Pro版へのアップグレード NEW in v3.0.0

Pro版（$1/月）にアップグレードすると、すべての機能を無制限で利用できます。

  1. C-kitを開く（拡張機能アイコンをクリック）
  2. 「Account」タブに移動
  3. Googleアカウントでサインイン
  4. 「🚀 Upgrade to Pro」ボタンをクリック
  5. Stripe Checkout画面で決済情報を入力
    💳 安全なStripe決済システムを使用
  6. 決済完了後、すぐにPro機能が利用可能になります！
